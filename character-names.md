# Character names

```
; -- semicolon
: -- colon
* -- asterisk
/ -- slash
\ -- backslash
- -- dash
_ -- underscore
() -- parenthesis
[] -- (square) brackets
{} -- braces
, -- comma
. -- dot
& -- ampersand
@ -- at
= -- equals
'' -- quotes
"" -- double quotes
| -- pipe
... -- ellipsis
! -- exclamation mark
? -- question mark
```
